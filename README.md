# kaggle_house_pricing
Predict house price when there are a lot of features in dataset
1. EDA, handle right skewed target by taking log, observed outliers
2. combine test and train data
3. Data Handling, check logic and fill null, remove extreme data that may affect model 
4. feature engineering
5. fix skewed numerical features
6. encoding and scaling
7. Train lasso and find coeffiect of feature
8. Train lasso with selected feature having coeffient >0
9. Train linear regression with all columns
10. Train linear regression with selected feature having coeffient >0 got from lasso
11. Model Evalutaion
